Materi Pembelajaran

üì• 1. INPUT (Masukan)

Definisi: Input adalah proses memasukkan data atau instruksi ke dalam sistem komputer melalui perangkat input.

Fungsi: Memberikan data mentah kepada komputer untuk diproses.

Perangkat Input: Keyboard, Mouse, Scanner, Microphone, Webcam, Touchscreen, Joystick

‚öôÔ∏è 2. PROCESS (Pemrosesan)

Definisi: Process adalah tahap pengolahan data yang dilakukan oleh CPU (Central Processing Unit).

Fungsi: Mengubah data mentah menjadi informasi yang bermakna melalui operasi aritmatika dan logika.

Komponen Utama:

  • CPU (Processor) - Otak komputer
  • ALU (Arithmetic Logic Unit) - Melakukan operasi matematika
  • Control Unit - Mengatur aliran data
  • RAM (Random Access Memory) - Penyimpanan sementara

üì§ 3. OUTPUT (Keluaran)

Definisi: Output adalah hasil pemrosesan data yang ditampilkan dalam bentuk yang dapat dipahami manusia.

Fungsi: Menampilkan informasi hasil pemrosesan kepada pengguna.

Perangkat Output: Monitor, Printer, Speaker, Proyektor, Headphone, Plotter

üíæ 4. STORAGE (Penyimpanan)

Definisi: Storage adalah media untuk menyimpan data dan informasi secara permanen atau sementara.

Fungsi: Menyimpan data, program, dan sistem operasi untuk penggunaan jangka panjang.

Jenis Storage:

  • Primary Storage: RAM, ROM, Cache
  • Secondary Storage: Hard Disk, SSD, Flash Drive
  • Tertiary Storage: Cloud Storage, Tape Backup

Contoh Penerapan

üéÆ Contoh 1: Bermain Game

  • INPUT: Menekan tombol keyboard/controller untuk menggerakkan karakter
  • PROCESS: CPU memproses perintah, menghitung posisi karakter, grafik, dan fisika game
  • OUTPUT: Monitor menampilkan pergerakan karakter, speaker mengeluarkan suara
  • STORAGE: Save game disimpan di hard disk/SSD

üìù Contoh 2: Menulis Dokumen

  • INPUT: Mengetik teks menggunakan keyboard
  • PROCESS: Aplikasi pengolah kata memproses teks, format, dan layout
  • OUTPUT: Teks ditampilkan di layar monitor
  • STORAGE: Dokumen disimpan di hard disk sebagai file .docx atau .pdf

üéµ Contoh 3: Mendengarkan Musik

  • INPUT: Mengklik tombol play pada aplikasi musik
  • PROCESS: CPU membaca file audio, decode format MP3/FLAC
  • OUTPUT: Speaker/headphone mengeluarkan suara musik
  • STORAGE: File musik tersimpan di hard disk atau streaming dari cloud

üñ®Ô∏è Contoh 4: Mencetak Dokumen

  • INPUT: Memberikan perintah print melalui mouse/keyboard
  • PROCESS: CPU mengolah data dokumen menjadi format yang dapat dicetak
  • OUTPUT: Printer mencetak dokumen di kertas
  • STORAGE: Dokumen asli tetap tersimpan di komputer
